<Updates>
  <Update label="January 2026">
    ### Modernization & Developer Experience

    * **Premium Landing Page**: Redesigned the hero section with a modern aesthetic, vibrant gradients, and smooth Framer Motion animations.
    * **Interactive Playground**: Launched a dedicated playground at `/playground` with document management features and autosave.
    * **Flexible Toolbar API**: Introduced `toolbarConfig` for granular control over toolbar items, custom styling, and variable positioning (fixed, sticky, or relative).
    * **Package Modernization**: Updated all documentation and examples to use the official `@collabchron/notiq` package name.
    * **Build & Search Improvements**: Resolved Orama search integration issues and optimized Next.js build configuration.
    * **UI Refinements**: Polished Excalidraw integration and added custom scrollbar support across the workspace.

  </Update>

  <Update label="May 2025">
    ### General Improvements

    * Redesigned notification settings for better control over email and in-app alerts.
    * Fixed issue where some users were not receiving 2FA codes via SMS.
    * Activity log now includes more granular details for API usage.
    * Added keyboard shortcuts to the dashboard for quicker navigation.

  </Update>

  <Update label="April 2025">
    ### Mobile Enhancements

    * Major performance optimizations for older Android devices.
    * Added gesture support for calendar rescheduling.
    * Resolved an issue causing data sync delays when switching between networks.
    * Push notification reliability improved for background updates.

  </Update>

  <Update label="March 2025">
    ### Integration Upgrades

    * Microsoft Teams integration now supports deep-linking to projects.
    * Added webhook event for task reassignment.
    * Jira integration now includes sprint sync and story point mapping.
    * New Zapier triggers for completed tasks and file uploads.

  </Update>
 </Updates>
