# Content Blocks

Notiq supports a wide variety of content blocks that go beyond simple text. 

## Media & Embeds
- **YouTube**: Embed videos by simply pasting the link or using the slash command.
- **Twitter**: High-quality tweet embeds.
- **Images**: Drag & drop or upload images with built-in resizing and alignment.

## Technical & Academic
- **Code Blocks**: Syntax highlighting for 20+ languages powered by Prism/Shiki.
- **Math Equations**: Render beautiful LaTeX equations using KaTeX.
- **Inline Code**: For technical mentions within paragraphs.

## Design & Visualization
- **Excalidraw**: Create hand-drawn diagrams directly inside your document.
- **Figma**: Interactive Figma prototypes and designs.

## Interactive Elements
- **Polls**: Create interactive polls for user feedback.
- **Collapsible Sections**: Keep documents organized with toggleable content.
- **Step Guides**: Create clearly numbered step-by-step processes.

## Layout & Organization
- **Multi-column**: Create complex layouts with resizable columns.
- **Tables**: Powerful table support with cell formatting.
- **Callouts**: Highlight important notes, warnings, or tips.

### Using Slash Commands
Most blocks can be inserted by typing `/` followed by the block name (e.g., `/code`, `/image`, `/math`).
